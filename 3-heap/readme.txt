This is a Min and Max 3-Heap with a terminal based menu the controls are as follows:

1. BuildHeap: Inserts an element at the end of linked list.
2. Insert: Deletes an element given if present 
3. Delete: Finds smallest element from the list
4. MinLevelElements: Finds largest element from the list.
5. MaxLevelElements: Finds average of all the elements of the list.
6. Exit: Exits the program
