This is a hash table with two types of closed hashing with a terminal based menu the controls are as follows:

1- Insert - Inserts an element into both hash table
2- Delete - Deletes an element from the hash table 
3- FindByName - Searches for the element in both table return position in both
4- FindByRating - returns all restaurants that has the given rating for both tables
5- SearchByPrice - returns all restaurants for the given price for both tables
6- Print - prints out both tables 
7- Exit - exits the program
