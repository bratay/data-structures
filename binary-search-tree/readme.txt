This is a binary tree with a terminal based menu, the controls are as follows:

AddItem – This function inserts the character element into the tree.
Delete – This function deletes the character from the binary search tree.
Leaf – This function will check whether or not x has a child.
PrintLeaf – This function prints the leaf nodes from left to right.
TreeHeight – Prints the height of the tree.
Preorder –  Traverses   the tree in pre-order and prints   all the characters.
Postorder – Traverses   the tree in post-order and prints all the characters.
Inorder – Traverses   the tree in in-order and prints   all the characters.
SearchElement – This function will search for the character.  If it finds the character, it should return true.  Otherwise, it should return false.
Exit - Ends the program.
