This is a binary tree with a terminal based menu, the controls are as follows:

1-IsFull - true if the tree is currently full
2-AddItem - inserts a number into the tree
3-Delete - removes a element from the tree
4-Leaf - true is input is a leaf node
5-PrintLeaf - prints all nodes in order from left to right
6-PrintTreeHeight - gives the tree height
7-Preorder - prints pre order traversal
8-Postorder - prints post order traversal
9-Inorder - prints inorder traversal
10-Levelorder -prints level order traversal
11-Exit - exits the program
