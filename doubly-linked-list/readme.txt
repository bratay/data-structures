This is a double linked list with a terminal based menu the controls are as follows:

1. Insert: Inserts an element at the end of linked list.
2. Delete: Deletes an element given if present 
3. Smallest: Finds smallest element from the list
4. Largest: Finds largest element from the list.
5. Average: Finds average of all the elements of the list.
6. Merge2Lists: Merges a new given list with the old list sorted in ascending order
7. Print: Prints all elements of the list.
8. exit: Exits the program
